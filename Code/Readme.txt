The project is coded in JAVA 
The running environment is jre 1.8.0_60 
IDE: Eclipse LUNA

Import the archived file 'projectCode.zip' into IDE & execute the code

For running the code:
	The main method is present in GraphTest.Java
	Additionally we are maintaining a config.properties files, that consist of the configurations like Pearson correlation coefficient 'r' & the path of the data. 
	For example r = 0.9
		    path = <systempath> // This is the path of the system where complete data is present in binary form
	This file is present in the project classPath root folder
